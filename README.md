<h1>🔐 Node.js Auth & Authorization API </h1>

A simple and secure REST API built with Node.js, Express, and MongoDB implementing JWT Authentication and Role-Based Authorization.

<hr>

<h1>🚀 Features </h1>
<ul>
 <li>User Registration & Login</li>
  <li>Password Hashing (bcrypt)</li>
  <li>JWT Authentication</li>
  <li>Role-Based Authorization (User / Admin)</li>
 <li>Protected Routes</li>
 <li>MongoDB Integration</li>
</ul>
